### Получение карточки произведения
```
GET /work/{id}
```
Ответ:
```
{
    work_id: Int,                     # id произведения
    last_modified: DateTime,          # время последнего изменения данных
    work_name: String,                # название произведения
    work_name_orig: String,           # оригинальное название
  …
}

```
